# 블록(block) 코드 강조
백틱기호 3개를 이용하여 코드 강조 가능
```html
    <a href="https://google.com">GOOGLE</a>
```
```css
    h1 {
      font-size: 12px;
    }
```
```plantext
  단순텍스트도 블록코드로 강조 가능
```
